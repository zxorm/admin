加载按钮（Loading buttons）
我们通常在点击按钮，提交ajax数据的时候，需要为按钮添加一个加载和引用效果，幸运的是，iSmartJs提供了非常简洁的方法实现这样的效果
加载按钮

代码

使用下面列出的类可以快速创建一个带有预定义样式的按钮。 

<button type="button" class="btn btn-info " >
    提交 <i class="fa fa-refresh "></i></button>
//icon 按钮
<button type="button" class="btn btn-orange btn-sm " s-click="return submit()">
    <i class="fa fa-refresh "></i>
</button>
<script type="text/javascript">
    
    function submit(){
        var deferred = $.Deferred();
        setTimeout(function(){
        deferred.resolve();
        }, 4000);
        return deferred;
    }
    
</script>
按钮点击想要获取加载效果，必须要返回一个Deferred对象，当该Deferred对象resolve、或者reject的时候，会自动回复原来的效果。 所以这样的调用方法适合iSmartJs的ajax函数、jQuery的ajax函数，因为这些方法都会返回一个Deferred对象。
