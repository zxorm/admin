确认框
确认框可以使用在诸如删除确认、提交确认、关闭确认，用于提示用户确认操作的时候。
调用方法
S.confirm(msg, [level,] [option])
参数名 类型 默认值 说明
msg String 显示的信息
level String info 提示的级别，可选值为info，warning，success，danger
level参数也可定义显示的标识（sign）,如success:fa fa-cloud-upload，通过冒号:分隔，success是显示级别， fa fa-cloud-upload是sign的class，用来自定义icon。
option Object 
{
    title: "提示",
    sureBtnName: "确定",
    cancelBtnName: "取消"
}
titleconfirm框的标题，btnNameconfirm框的确定按钮的显示文字。
警告框

自定义图标代码

S.confirm('您确定提交数据吗', 'warning:fa fa-cloud-download')

完整代码

S.confirm("您确定要提交数据吗？", level).done(function(){
    S.alert("您点击了确认按钮");
}).fail(function(){
    S.alert("您点击了关闭按钮");
})
通过自定义class来自定义显示级别
使用level参数值定义class.s-confirm-{level}，就可以实现自定义显示级别。 如S.confirm('您确定提交数据吗？', 'dark')
