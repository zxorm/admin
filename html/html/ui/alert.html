Alert
警告框提供一个便捷的信息提示功能，用来替代原生的window.alert
调用方法
S.alert(msg, [level,] [option])
参数名 类型 默认值 说明
msg String 显示的信息
level String info 提示的级别，可选值为info，warning，success，danger
level参数也可定义显示的标识（sign）,如success:fa fa-cloud-upload，通过冒号:分隔，success是显示级别， fa fa-cloud-upload是sign的class，用来自定义icon。
option Object 
{
    title: '提示',
    btnName: '确定'
}
titlealert框的标题，btnNamealert框的确定按钮的显示文字。
警告框

自定义图标代码

S.alert('恭喜你，下载完成！', 'warning:fa fa-cloud-download')

回掉示例

S.alert('恭喜你，下载完成！', 'warning:fa fa-cloud-download').done(function(){
    S.toast('您关闭了警告框Alert');
})
通过自定义class来自定义显示级别
使用level参数值定义class.s-alert-{level}，就可以实现自定义显示级别。 如S.alert('iSmartJs', 'dark') 

<style>
        .s-alert{

        }
        /**
            定义级别的符号
        */
        .s-alert.s-alert-dark .s-alert-sign.s-alert-sign-default{
            font-family: 'Pe-icon-7-stroke';
        }
        .s-alert.s-alert-dark .s-alert-sign.s-alert-sign-default:before{
            content: "\e612";
            color: #000;
            font-weight: bold;
        }
</style>
