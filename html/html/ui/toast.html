Toast通知
toast可以作为错误通知提示，消息通知提示等使用场景
调用方法
S.toast(msg, level, holdTime)
msg 需要显示的信息
level 信息显示的级别，可以设定为：primary、info、success、danger、warning，violet，orange 默认为primary级别。
level参数也可定义显示的标识（sign）,如success:fa fa-cloud-upload，通过冒号:分隔，success是显示级别， fa fa-cloud-upload是sign的class，用来自定义icon。
holdTime 持续时间，单位为毫秒，默认为3000，如果需要stick效果，则可以将该值设置为无限大

自定义显示sign
使用level参数定义显示的标识（sign）,如success:fa fa-cloud-upload，通过冒号:分隔，success是显示级别， fa fa-cloud-upload是sign的class，用来自定义icon。
代码：S.toast('文件已经成功上传到云服务器！', 'success:fa fa-cloud-upload')

自定义Toast显示的位置
通过定义.s-toast-container的css样式实现Toast的显示位置，如显示在页面顶部中间可以如下定义：
<style>
    .s-toast-container{
        position: fixed;
        width: 500px;
        left: 50%;
        top: 60px;
        margin-left: -250px;
        bottom: auto;
    }
</style>
通过自定义class来自定义显示级别
使用level参数值定义class.s-toast-{level}，就可以实现自定义显示级别。 如S.toast('iSmartJs', 'dark') 

<style>
        .s-toast-dark{
            background: #000;
            color: #fff;
        }
        /**
            定义级别的符号
        */
        .s-toast.s-toast-dark .s-toast-sign.s-toast-sign-default{
            font-family: 'Pe-icon-7-stroke';
        }
        .s-toast.s-toast-dark .s-toast-sign.s-toast-sign-default:before{
            content: "\e612";
        }
</style>
