列表控件
使用列表控件、行选择控件、分页控件、表单控件可以构建出查询列表。
Follow Me
列表控件最常用的地方是和行选择控件、分页控件、表单搜索控件组合起来构建查询列表，iSmartJs强大的控件渲染与协作机制可以使得查询列表可以实现完全的自定义。

  1. 一个简单的列表

    • 声明loop控件，并绑定数据源

      使用s='loop'将tbody声明为loop控件，并且通过s-data="S.get('json/singers.json')"绑定loop控件的数据源。

    • 声明列表控件循环的行

      使用s-loop-role="row"声明列表控件的行，列表控件构建的时候，会根据该行的dom进行构建，可以声明多个行。 在这里，通常情况下，还需要声明行为datac控件，因为进行数据绑定的是行下面的用于显示数据的控件。

  2. 声明空数据提示行

    通常的，如果查询的列表数据为空的时候需要在列表中提示，比如提示：未查询到相关数据等，我们可以通过声明s-loop-role="empty" 来声明空数据提示行，当查询数据列表为空的时候，该行会作为提示行显示出来。

  5. 分页列表

    分页列表需要通过分页控件配合列表控件来构建。简单来说，就是通过datac控件把查询的返回值分别赋值给列表控件和分页控件，点击分页控件页数的时候， 将当前的页数保存到查询form当中再次进行查询，即可实现分页查询功能。
    歌手 地区 歌曲 排行榜
    未查询到数据
    总记录数：

    代码

    <form id="searchForm2"
      
      s="form"
      s-form="{action: function(){ return $singersGrid.refresh();}}"
      
      class="form-inline space-v">
    <input type="text" name="name" class="form-control" placeholder="输入歌手名进行查询" />
    <input type="hidden" name="page" id="page" />
    <input type="submit" class="btn btn-primary btn-sm" value="查询" />
</form>
<table class="table table-striped"
       id="singersGrid"
       s="datac"
       s-data="S.get('json/singers-page.php', $searchForm2.node.serialize())"
        >
    <tr>
        <th>歌手</th>
        <th>地区</th>
        <th>歌曲</th>
        <th>排行榜</th>
    </tr>
    <tbody s="loop" s-data-filter="'results'">
    <tr s-loop-role="empty">
        <td colspan="4">
            <div class="space-v text-center">
                <i class="fa fa-info-circle text-info"></i> 未查询到数据
            </div>
        </td>
    </tr>
    <tr s-loop-role="row" s="datac">
        <td s s-data-filter="'name'"></td>
        <td s s-data-filter="'area'"></td>
        <td s s-data-filter="'song'"></td>
        <td s s-data-filter="'rank'"></td>
    </tr>
    </tbody>
    <tfoot>
    <tr>
        <td colspan="4">
            <span class="pull-right">总记录数：<span s s-data-filter="'total'"></span></span>
            <ul s="pagination"
                s-pagination="{action: function(){$page.node.val(this);$singersGrid.refresh();}}"
                class="pagination pagination-sm pull-right">
            </ul>
        </td>
    </tr>
    </tfoot>
</table>

    在这里使用了form控件、loop控件、pagination控件、datac控件的组合构建出了拥有查询功能的分页列表， datac控件负责把数据源返回的值分别赋值给loop控件和pagination控件，用以显示，当然也可以声明 <span s s-data-filter='total' ></span>用于显示总记录数。

<script type="text/javascript">
    function getCheckedRows(){
        var datas = $singersGrid2.getCheckedData();
        if(datas.length){
            S.alert(JSON.stringify(datas));

            S.toast("请先选择数据！");
        }

    check控件使用方法请参考check控件。

API
方法名 返回值 说明
empty() 清空
setRows(datas) 为控件赋值，基本上等同于默认apidata(datas)，调用该方法会清空现有的数据。
addRow(rowData, mode, indentNum) 添加一个新行。参数分别为：rowData行数据，mode行追加方式，默认为append，indentNum行缩进值，默认为0。
addRows(datas, mode, indentNum) 添加行数据。参数分别为：datas行数据，mode行追加方式，默认为append，indentNum行缩进值，默认为0。
hideAssistRows() 清理辅助行，比如空数据提示行、准备提示行、加载提示行等。
