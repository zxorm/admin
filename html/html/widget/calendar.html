  1. 一个基本的日期选择器

    代码

    <form class="form-inline">
    <input type="text" class="form-control" s="datetimepicker" style="width: 200px;" />
</form>

  2. 带时间选择的日期选择器

    代码

    <form class="form-inline">
    <input type="text" class="form-control" s="datetimepicker"
           s-datetimepicker="{minView:'hour', format:'yyyy-mm-dd hh:ii', pickerPosition:'top-left'}" style="width: 250px;" />
</form>
    更多配置参数请参考github：https://github.com/smalot/bootstrap-datetimepicker
参数配置
参数配置可参考https://github.com/smalot/bootstrap-datetimepicker， 通过s-datetimepicker属性来配置参数。
默认参数名 类型 默认值 说明
format String yyyy-mm-dd 日期的格式
autoclose boolean true 当日期选择后是否关闭
language String zh-CN 语言
minView String month 日期时间选择器所能够提供的最精确的时间选择视图。
todayHighlight boolean true 是否高亮显示当前日期
