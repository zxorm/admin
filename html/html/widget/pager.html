翻页
翻页控件只有两个按钮，一个为往前翻页的按钮用class="s-pager-prev"来定义，一个是往后翻页的按钮用class="s-pager-next"来定义。
  • 上一页
  • 下一页

代码：

<nav>
    <ul id="pager" s="pager" s-pager="{action: pageSelect}" s-data="pagerData" class="pager" >
        <li><a class="s-pager-prev" href="javascript:;">上一页</a></li>
        <li><a class="s-pager-next" href="javascript:;">下一页</a></li>
    </ul>
</nav>
<script type="text/javascript">
    var pagerData = {
        "total": 323,
        "pageSize": 10,
        "page": 1
    };
    function pageSelect(page, $pager){
        pagerData.page = page;
        $pager.data(pagerData);
    }
</script>

参数配置

参数名 类型 默认值 说明
action Function 翻页项点击时的方法，该方法的this引用指向翻页的页数，如当点击第2页的时候，该方法的this对象值就是2
disabledClass String disabled 按钮禁用状态的class
pageKey String page
pageSizeKey String pageSize
totalKey String total
