分页
在iSmartJs中，我们需要s="pagination"将该组件声明为分页控件，添加.pagination-sm、.pagination-lg来定义尺寸

代码：

    <ul s="pagination"
       s-pagination="{action:pageSelect}"
       s-data="paginationData"
       class="pagination pagination-sm">
    </ul>
    <script type="text/javascript">
        var paginationData = {
            "total": 323,
            "pageSize": 10,
            "page": 1
        };
        function pageSelect(page, $pagination){
            paginationData.page = page;
            $pagination.data(paginationData);
        }
    </script>

参数配置

参数名 类型 默认值 说明
action Function 分页项点击时的方法，该方法的this引用指向分页的页数，如当点击第2页的时候，该方法的this对象值就是2
preText String < 上一页按钮的显示文字
nextText String > 下一页按钮的显示文字
startText String « 前一页的按钮的显示文字
endText String » 后一页的按钮的显示文字
disabledClass String disabled 按钮禁用状态的class
activeClass String active 按钮激活状态的class
pageKey String page
pageSizeKey String pageSize
totalKey String total
showSize int 11 显示的总页数
